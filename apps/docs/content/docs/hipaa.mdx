## Controls

**Session Security** — 15-min idle timeout, 8-hour absolute. JWT with secure HttpOnly cookies.

**Encrypted PHI** — AES-256 with versioned keys. Rotation without downtime.

**BAA Framework** — Templates and workflows ready for enterprise vendor agreements. 7 vendor BAAs pending signature (DigitalOcean, Upstash, Anthropic, Deepgram, Sentry, Resend, Twilio).

**De-identification** — Presidio-based PII detection and stripping for research data sharing.

## Strategic Position

Per the board strategic memo (March 2026), the US/Canada market is a **NO-GO** for the core clinical workflow. Cortex's HIPAA readiness is positioned for selling a headless "Hallucination Firewall API" to existing US vendors—not direct clinical deployment.

LATAM (Brazil first, then AR/BO) is the aggressive go-to-market. LGPD compliance is the primary compliance surface; HIPAA is maintained for API licensing opportunities.
