Class I CDS classification. Deterministic logic avoids the SaMD trap.

## Why This Matters

By using **deterministic JSON-Logic rules** for safety-critical decisions, Cortex qualifies as Class I CDS under ANVISA RDC 657/2022. This avoids SaMD classification requiring multi-year clinical trials.

Competitors using LLMs for clinical recommendations face Class II/III registration: clinical trials, conformity assessment, 12–24 month timelines.

## The Deterministic Guarantee

All prescribing, dosing, and compliance checks use:
- JSON-Logic rules (not ML models)
- Full provenance metadata on every rule
- Source authority, evidence tier, citation URL, last-reviewed date
- 100% auditable, 100% explainable output

## Marketing Copy Audit (RUTH-enforced)

All customer-facing copy is audited. These words are **never** used in patient-facing text:

- diagnose
- detect
- prevent
- treat
- cure
- predict disease

Using any of these terms risks reclassification to Class II/III SaMD. This is a RUTH veto invariant—any copy containing these terms is blocked before shipping.
