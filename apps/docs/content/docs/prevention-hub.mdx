## Seven Health Domains

| Domain | Focus |
|--------|-------|
| Cardiometabolic | Hypertension, diabetes, lipids, DOAC safety |
| Oncology | Cancer screening, follow-up, biomarkers |
| Musculoskeletal | Osteoporosis, fall risk, joint health |
| Neurocognitive | Cognitive screening, depression, dementia |
| Gut Health | GI screening, microbiome, nutrition |
| Immune | Vaccines, autoimmune, infection prevention |
| Hormonal | Thyroid, reproductive, metabolic hormones |

## Protocol Sources

WHO · NHS · ESC · RACGP · USPSTF · 50+ Protocols

All protocols trace to Tier 1 (international guidelines) or Tier 2 (peer-reviewed meta-analyses). Tier 3 (expert opinion) is flagged but never used for automated recommendations.

## Screening Workflow

**Protocol Rule → Patient Match → Reminder (DUE) → Order / Referral**

Statuses: DUE → SCHEDULED → COMPLETED (or OVERDUE).

WebSocket streaming delivers real-time alerts when lab results shift a patient's risk profile.

## Risk Scoring

AI-powered risk scoring (0–100 per domain) identifies patients drifting toward adverse outcomes before symptoms present. Collaborative clinical templates with version control enable multi-provider care coordination.
