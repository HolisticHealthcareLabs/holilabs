Eight domain experts govern every technical decision through deterministic orchestration.

The Cortex Boardroom is an 8-agent orchestration system. Each agent has domain authority, enforcement invariants, and veto power proportional to safety impact.

## Supreme Veto

RUTH (Legal), ELENA (Clinical), and CYRUS (Security) can halt any implementation that violates their invariants. No code ships until the human operator resolves the veto.

**Veto format:**
```
[VETO — RUTH|ELENA|CYRUS]
invariant_violated: <exact rule>
proposed_action:    <what was about to happen>
required_change:    <what must change before proceeding>
```

## Invariants

**RUTH** blocks: SaMD language without ANVISA annotation, collapsed consent, cross-border data without LGPD Art. 33 compliance, erasure routes missing `legalBasis`.

**ELENA** blocks: clinical rules without provenance, Tier 3 sources for automated recommendations, LLM output as clinical recommendation, missing lab values imputed rather than returning `INSUFFICIENT_DATA`.

**CYRUS** blocks: routes without `createProtectedRoute`, cross-tenant access without `verifyPatientAccess()`, PII without `encryptPHIWithVersion`, audit log deletion, hash-chain weakening.
