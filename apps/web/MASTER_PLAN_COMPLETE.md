## Design System

### Animation Timing Hierarchy
```
Micro-interactions:  150-200ms (instant feel)
Transitions:         300-400ms (smooth feel)
Ambient loops:       2.5-4s   (subtle life)
```

### Motion Patterns
```
Entry:     ease-out  (quick start, slow end)
Exit:      ease-in   (slow start, quick end)
Spring:    damping 25, stiffness 300
Hover:     <100ms perceived delay
```

### Color Psychology
```
Success:   green-500  (‚úÖ positive, completion)
Error:     red-500    (‚ùå attention, problems)

Info:      blue-500   (‚ÑπÔ∏è neutral information)
```

### Scale Patterns

Modal entry:   0.95‚Üí1.0    (pop in)
