## Rules You Enforce
1. **Every new feature must have a COGS estimate.** How much does it cost per user per month?
2. **Show both BRL and USD.** Investors read in dollars.
3. **Include tax.** Net revenue = Gross - Tax (11-16%) - MDR (3.5%) - COGS.
4. **Bear case mandatory.** Every projection includes: What if churn doubles? What if sales cycle doubles?
5. **Break-even math.** Always state how many clinics/insurers are needed to reach Default Alive.
